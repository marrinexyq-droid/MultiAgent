# Image Analysis Checklist

## Record These Fields

- image width and height
- long-page vs viewport screenshot
- page category
- top navigation composition
- hero structure
- section order
- dominant colors
- heading font feeling
- body text density
- border and radius system
- card types
- CTA variants
- banner or marquee usage
- illustration style
- icon style
- footer structure

## Note These Risks

- blurry or unreadable text
- cropped hero or footer
- overlapping cards that depend on scroll animation
- hidden states that cannot be inferred from a static image
- mobile behavior not visible in desktop screenshot

## Output Shape

Produce a concise brief with:

- `summary`
- `sections`
- `style_tokens`
- `interaction_hints`
- `unknowns`
