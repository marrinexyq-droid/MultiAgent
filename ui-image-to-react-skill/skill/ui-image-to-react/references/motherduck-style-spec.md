# MotherDuck Style Spec

## Design Direction

Use a "playful-industrial" SaaS landing-page aesthetic: retro-technical monospace headings, warm cream surfaces, strong charcoal borders, and approachable hand-drawn illustration cues. Aim for friendly data infrastructure, not sterile enterprise minimalism.

## Palette

- background: `#F4EFEA`
- text and borders: `#383838`
- primary CTA: `#6FC2FF`
- banner yellow: `#FFDE00`
- mint teal: `#53DBC9`
- coral red: `#FF7169`
- deep teal: `#16AA98`
- hover blue: `#2BA5FF`
- footer background: `#383838`
- white cards: `#FFFFFF`
- disabled text: `#A1A1A1`
- disabled background: `#F8F8F7`

## Typography

- headings: monospace look, uppercase, `font-weight: 400`
- body: clean sans-serif like Inter, `font-weight: 300` for larger body copy
- H1 target: `56px`, line-height `1.2`, slight tracking
- H2 target: `32px` to `40px`, uppercase
- body: `18px` to `20px`
- nav and buttons: uppercase, `14px` to `16px`

## Borders And Radius

- use `2px solid #383838` widely
- keep corners nearly square with `2px` radius max
- use divider lines between major sections

## Layout

- content width: `1302px`
- horizontal padding: `30px`
- generous section spacing
- sticky or fixed navigation
- long-form landing page rhythm with full-width banners between content sections

## Components

- buttons use offset-shadow framing instead of soft drop shadows
- cards use white fills and charcoal borders
- section labels can float as dark tags with white uppercase text
- marquees should feel editorial and mechanical, not flashy

## Decorative Language

- cloud forms
- hand-drawn lines or mascot-like illustration energy
- playful diagrams mixed with product-oriented cards and blocks

## Do Not Use

- glassmorphism
- pill buttons
- bold display headings
- pure white page backgrounds
- pure black UI
- blurred surfaces
- flashy 3D effects
